title: 命令给安卓程序打包

---

在命令行，进入到项目（project，一定是project，而不是Module）对应的目录。

1. 先执行改变gradle的权限。

   ```shell
   chmod +x gradlew
   ```


2. 然后运行打包

   ```shell
   ./gradlew assembleRelease
   ```
   <!-- more -->

3. 出现`BUILD SUCCESSFUL`后安装到手机

   ```shell
   adb install <apk路径>
   ```
